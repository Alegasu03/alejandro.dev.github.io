# Portfolio de Alejandro

Landing personal sencilla para GitHub Pages con información profesional, proyectos y enlaces de contacto.

## Contenido

- Cabecera con presentación y accesos rápidos.
- Sección "Sobre mí" con datos básicos.
- Sección de proyectos destacados.
- Footer con enlaces sociales.

## Ejecutar en local

```bash
python3 -m http.server 8000
```

Después abre `http://localhost:8000`.
